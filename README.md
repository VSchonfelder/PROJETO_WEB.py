# Projeto 02 - Mini-projeto Web utilizando PostgreSQL
 

Vinícius Schonfelder


Aqui vai a explicação do projeto, como ele funciona e o que foi feito nele desde o começo até o fim dele. Inicialmente, foi instalado o sistema gerenciador de banco de dados PostgreSQL, e foi criado e configurado em sequência o banco de dados 'pdv', utilizando a ferramenta pgAdmin. Após isso, foi feita a instalação do Python na sua versão mais recente, e já dentro do terminal de prompt de comando, foi criado um ambiente virtual dentro da pasta 'Projeto2', que é a pasta do trabalho, com o comando python -m venv venv. Em seguida o ambiente virtual foi ativado com o comando venv\Scripts\activate. Após essa etapa, foi necessário instalar as dependências documentadas no arquivo requirements.txt, e os pacotes foram instalados utilizando o comando pip install -r requirements.txt. Após isso o próximo passo foi criar e modificar arquivo app.py, que é o arquivo principal da aplicação Flask, que configura o servidor web e se conecta ao banco de dados PostgreSQL. Nele também foi adicionada uma mensagem de erro pra caso o comando SQL digitado esteja com erro, e em seguida é permitido ao usuário digitar o comando novamente sem travar a aplicação. O outro arquivo obrigatório e essencial para a aplicação é o index.html, que é basicamente o template principal da aplicação, responsável pela interface do usuário, ele permite a entrada de comandos SQL e uma tabela para exibir pro usuário os resultados de suas consultas com os comandos SQL. Sobre o arquivo styles.css, ele basicamente define a aparência dos elementos HTML, estilizando a aplicação, o que inclui layouts, cores, fontes e etc. Esse arquivo é referenciado pelo index.html, o que define como visualmente ficará os elementos HTML da aplicação, como a tabela de resultados por exemplo. Para fazer a aplicação inicializar, é necessário executar um comando no terminal dentro da pasta do trabalho e com o ambiente virtual já ativo, o comando é python app.py. Após isso, o usuário acessa a aplicação web com localhost:5000 e insere um comando SQL na aplicação, assim o app.py recebe o comando SQL, se conecta ao banco de dados PostgreSQL, faz a consulta e exibe os resultados obtidos pro usuário na tabela, que terá aparência definida pelo styles.css.
